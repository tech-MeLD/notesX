# 安装 NodeJS

包含 `npm` `npx` ，使用 `winget` 安装

```bash
winget install OpenJS.NodeJS.LTS
node -v
```

# 安装 pandoc

[参考文档](https://github.com/jgm/pandoc/blob/main/INSTALL.md)

## Windows 平台

使用 `winget` 安装

```shell
winget install --source winget --exact --id JohnMacFarlane.Pandoc
```

## MacOS 平台热爱

使用 `Homebrew` 安装

```shell
 brew install pandoc
```

## Linux 平台

检查您的软件包管理器中的 pandoc 版本是否过时。Pandoc 已收录于 [Debian](https://packages.debian.org/search?keywords=pandoc)、[Ubuntu](https://packages.ubuntu.com/search?keywords=pandoc)、[Slackware](https://www.slackbuilds.org/result/?search=pandoc&sv=)、 [Arch](https://archlinux.org/packages/?q=pandoc)、[Fedora](https://packages.fedoraproject.org/pkgs/pandoc/pandoc/)、[NixOS](https://search.nixos.org/packages?query=pandoc)、[openSUSE](https://software.opensuse.org/package/pandoc)、[gentoo](https://packages.gentoo.org/package/app-text/pandoc) 和 [Void](https://voidlinux.org/packages/?arch=x86_64&q=pandoc) 软件仓库中

## Docker 平台

[pandoc 的官方 Docker 镜像可以在https://github.com/pandoc/dockerfiles](https://github.com/pandoc/dockerfiles) 和 [dockerhub](https://hub.docker.com/) 上找到 。
pandoc [/core](https://hub.docker.com/r/pandoc/core) 镜像包含 `pandoc`。
pandoc [/latex](https://hub.docker.com/r/pandoc/latex) 镜像还包含使用 pandoc 生成 PDF 所需的最小 LaTeX 安装。
要使用 Docker 运行 pandoc，请转换 `README.md` 为 `README.pdf`

```shell
docker run --rm --volume "`pwd`:/data" --user `id -u`:`id -g` pandoc/latex README.md -o README.pdf
```
