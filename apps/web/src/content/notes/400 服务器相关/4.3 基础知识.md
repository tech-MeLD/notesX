# 网络协议分层 

|层级|协议|
|---|---|
|应用层|HTTP / SSH / FTP|
|传输层|TCP / UDP|
|网络层|IP / ICMP|

TCP / UDP 才有端口，ICMP 没有端口，ICMP 是网络层协议，它不经过 TCP 或 UDP。
