# Easy typing

书写简易化，适合中文写作，功能全面
符号自动配对及删除
选中文本替换
两次按下实现全半角切换，不用频繁切换输入法

# Linter

格式化文档

# Annotator

PDF 阅读器

# Clear unused images

一键清除未链接的图片

# Pandoc Plugin + Export  settings

导出为其他各种格式 `Export to ...`，markdown、pdf、word ...

# Footnotes shortcut + Linter

结合 Linter 使用
快速脚注 `alt + 0` （常用）、`alt + -`

# Custom Attachment location +Consistent Attachments and Links

1、自定义附件位置，重命名附件
2、将图片 wiki 链接转换成标准的 markdown 相对路径链接

# Image auto upload + picgo + github

自动上传图床到云端（GitHub）

# Commander

指令自定义显示

# Image converter

不适合使用
自定义缩小放大图片，会导致文件图片格式新增命名，但不影响图片路径，不适合和 Custom Attachment location 混用，会影响该插件的生成附件逻辑
