# 引用和块引用

### 标题引用

- [[外部文档名#一级标题#...
不支持文件夹引用，只支持文件夹下的子文档，即具体的 md 文件，继续输入 `#` 可链接下级标题
- [[#当前文档中一级标题#...
适用于当前文章内容引用，继续输入 `#` 可链接下级标题，引用的本文档内容在出链列表不显示

### 块引用

- [[#^...
本文档块引用标准格式
- [[^...
本文档块引用简略格式，最终显示会变成标准格式
- [[^^引用的内容块
全局块引用

块引用尽量使用全局引用，因为更直观。可以通过观察右侧栏出链列表，能轻松发现引用外部文档会有标注，引用当前文档则没有标注

# 快捷键

`ctrl + o` 全局模糊查询
`ctrl + p` 命令
`ctrl + enter` 快速新建笔记
`ctrl + shift + enter`

`[[]]` 新建笔记（不管存在与否），`ctrl + enter` 新建笔记
`ctrl + o` 新建笔记（不存在），`enter` 新建笔记
`ctrl + o` 新建（存在），`ctrl + shift + enter` 新建笔记
`ctrl + p` 输入 `模版`，可以快速插入自定义的Templates，

`ctrl + F` 当前文档搜索
`ctrl + shift + F` 全局搜索
