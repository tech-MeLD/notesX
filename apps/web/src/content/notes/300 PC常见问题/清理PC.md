# 清理个人文件

## Windows
`win+E` 在桌面、下载、文档等文件夹中选择需要删除的文件，`shift+delete` 彻底删除文件

## Mac
`Command+option+delete` 彻底删除

# 清理应用中的个人数据

## Windows
微信：信设置 ->账号与存储 ->清空全部聊天记录，复制存储路径，完全退出微信后，在文件存储器中复制路径，将内容 `shift+delete` 彻底删除

Wps：点击账号头像 ->退出账号，弹出提示中选择清除数据并删除登录记录

卸载其他软件：Geek
## Mac
微信：在微信中随便找到一个视频文档等文件，右键 ->在 finder 中显示，然后跳转到 `微信-Data`，`Command+option+delete` 彻底删除

卸载其他软件：clean my Mac

# 清理浏览器数据

## Edge

账号设置 ->删除个人资料

## Chrome

头像 ->管理 chrome 个人资料 ->删除账号

## Safari

用户场景，其他类似


# 清理系统账号

## Windows

`win+i`，左上角头像账号，再点击你的账户，找到你的账号，点击账户设置 ->管理
或者直接打开 https://account.Microsoft.com

点击设备, 找到你的设备，`删除设备`，弹出 `已删除设备` 后，选择从 `microsoft store 删除`，`断开链接`，回到桌面账户设置页面

![](attachments/清理PC/file-20260306142132207.png)

![](attachments/清理PC/file-20260306142132213.png)

设置好后，注销并完成，再次登录可以看到微软账号已经退出，再次点击如下删除你的账号即可

![](attachments/清理PC/file-20260306142132219.png)

账号退出后，在隐私安全 ->搜索中，清楚所有搜索历史记录即可

![](attachments/清理PC/file-20260306142132223.png)

![](attachments/清理PC/file-20260306142132228.png)

## Mac

打开设置，左上角头像，退出登录就好
